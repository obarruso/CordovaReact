# Cordova with React template

Cordova version: 12.0.0

React version: 16.9.0

## Usage

### Clone

```shell
$ git clone https://github.com/obarruso/CordovaReact.git
```

### Install dependencies

```shell
$ cd CordovaReact
```

```bash
$ npm install
```

### Add a plataform

Ex.: android
```shell
$ cordova plataform add android
```

### Run application

```shell
$ cordova run
```
